# Research Summary

## Stack Recommendations

- **Python 3.10+** - Built-in only (hashlib, json, pathlib)
- **Storage**: File-based JSON (Obsidian vault compatible)
- **No external dependencies** for core functionality

## Table Stakes Features

1. Entity-predicate-value fact storage with timestamps
2. Session-scoped memory with TTL
3. Conflict detection via entity-predicate matching
4. Wedderburn-Kasczinski temporal resolution

## Differentiators

- Ring-theoretic session decomposition
- Ideal-theoretic forgetting (quotient rings)
- Module-valued confidence weighting
- Tensor product fact relationships

## Key Insight

The middleware is a **thin wrapper** over proven patterns from the existing Dilmun Protocol implementation. The innovation is in the ring-theoretic mathematics, not the storage layer.
