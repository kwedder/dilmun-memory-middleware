# Rust Ownership - Teaching for Children

## Core Concept: Ownership
In Rust, every value has ONE owner (variable).
When the owner goes out of scope, the value is dropped.

## Simple Rules:
1. Each value has one owner
2. Values can be moved (not copied)
3. References let you use values without taking ownership
4. Mutability must be declared

## Teaching Example:
```
let s1 = String::from("hello");  // s1 owns "hello"
let s2 = s1;                      // s1 loses ownership, s2 owns it
// println!("{}", s1);            // ERROR! s1 no longer owns it
```

## For Children:
- Think of ownership like a toy
- Only one child can own it at a time
- You can lend (borrow) it to friends
- When you give it away, you no longer have it
